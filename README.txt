Управление сдвиговыми регистрами 74HC595, 74HC156.
Реализация на Wiring (C/C++) для Arduino.

В каждой папке находятся отдельные скетчи.
Схема подключения прилагается.
В файле info.docx имеется подробное описание.

inreg.ino
Отправка в терминал состояний 4 тактовых кнопок, подключенных через входной сдвиговый регистр 74HC165.

outReg.ino
Отображение информации на 4-разрядном, 7-сегментном LED-индикаторе с общим анодом. Подключение через 2 сдвиговых регистра 74HC595N.

countInd.ino
Изменение значений разрядов семисегментного индикатора по нажатию на тактовые кнопки. Подключение кнопок и индикатора осуществляется через сдвиговые регистры.
